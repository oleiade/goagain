Quick Start

Docker Run locally

docker run -p 8080:8080 ghcr.io/oleiade/goagain:latest

REST API Search for cards

curl "https://api.goagain.dev/v1/cards?name=Strike&class=Warrior&limit=5"

REST API Get card details

curl "https://api.goagain.dev/v1/cards/enlightened-strike-red"

MCP Add to Claude Desktop 

{
  "mcpServers": {
    "flesh-and-blood": {
      "url": "https://mcp.goagain.dev/"
    }
  }
}

API Endpoints

Endpoint Description
GET /v1/cards Search cards with filters (name, type, class, set, pitch, keyword, q, legal_in)
GET /v1/cards/{id} Get card by unique ID or name
GET /v1/cards/{id}/legality Get card legality across all formats
GET /v1/sets List all card sets
GET /v1/sets/{id} Get set details with cards
GET /v1/keywords List all game keywords
GET /v1/keywords/{name} Get keyword description
GET /v1/abilities List all card abilities
GET /openapi OpenAPI specification (JSON)
GET /health Health check with data statistics

MCP Tools

The MCP server provides 9 tools for AI assistants to query Flesh and Blood card data, search by text, check format legality, and more.

View all MCP tools
Tool Description
search_cards Search cards by name, type, class, keywords, or other attributes
get_card Get full details of a specific card by unique ID or name
search_card_text Search for cards by text in their abilities or effects
get_format_legality Check a card's legality status across all formats
list_sets List all Flesh and Blood card sets
search_sets Search for sets by name or code
get_set Get details of a specific set including its cards
list_keywords List all game keywords with their explanations
get_keyword Get the description of a specific keyword
